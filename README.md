# Prettify JSON

pretty print json files

# Quickstart


Example of script launch on Linux, Python 3.5:

```#!bash

$ python pprint_json.py
input data path: C:\...\bars.json  # input full path
# output example
[
    {
        "global_id": 20660594,
        "system_object_id": "000069302",
        ...
        ...
        ...
        }
    }
]


```
